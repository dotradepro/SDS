# SDS-
Smart Device Simulator (SDS) — це локальний сервіс, що запускається в Docker, який імітує розумні пристрої в реальній мережі, використовуючи реальні протоколи. Мета — дозволити розробнику додавати віртуальні пристрої через вебінтерфейс і одразу тестувати взаємодію з ними через Selena без потреби у фізичних пристроях.
